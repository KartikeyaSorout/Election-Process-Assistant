# 🗳️ Election Process Assistant

An AI-powered chatbot that helps users understand the **US election process** — from voter registration and polling, to vote counting and result certification.

## ✨ Features
- 💬 Conversational AI assistant powered by LLaMA 3.3 70B
- 🕓 Chat history saved with Firebase Firestore
- 📊 Election timeline & voter turnout charts (Google Charts)
- 🔒 Secure API proxy via Cloudflare Workers
- ♿ Full accessibility support (ARIA, screen reader)
- 📱 Mobile responsive

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| AI Model | LLaMA 3.3 70B via Groq API |
| Backend | Cloudflare Workers |
| Database | Firebase Firestore |
| Analytics | Firebase Analytics |
| Charts | Google Charts |
| Hosting | GitHub Pages |
